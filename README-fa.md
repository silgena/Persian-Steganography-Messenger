# پیام‌رسان رمزنگاری پنهان

یک ابزار امن پایتون که پیام‌های انگلیسی شما را  رمزگذاری کرده و آن‌ها را به صورت کلمات فارسی نمایش می‌دهد. این برنامه از **رمزنگاری دو مرحله‌ای** استفاده می‌کند: یک فایل کلید اختصاصی و یک رمز عبور که فقط شما و گیرنده می‌دانید.

---

###  پیش‌نیازها 
ابتدا پایتون را نصب کنید و سپس کتابخانه‌های مورد نیاز را با دستور زیر نصب نمایید:

```bash
pip install -r requirements.txt
```
###  راه‌اندازی اولیه
قبل از ارسال پیام، باید "فایل کلید" اختصاصی خود را بسازید:

 اسکریپت زیر را اجرا کنید.
```bash
 python generate_key.py 
```
 فایلی به نام key.pem ساخته می‌شود.

 نکته بسیار مهم: این فایل را به دست دوستتان برسانید. هر دو نفر باید دقیقاً از یک فایل کلید استفاده کنید.

###  نحوه رمزگذاری و ارسال پیام
 پیام خود را در فایل english_text.txt بنویسید و ذخیره کنید
 
برنامه را اجرا کنید:
```bash
 python secret_message.py 
```
رمز عبور انتخابی (مثلاً: ERFAN) را وارد کنید.

  گزینه 1 (Encrypt) را انتخاب کنید.
  
متن فارسی در فایل persian_text.txt ایجاد می‌شود.

این متن را برای دوست خود بفرستید.


###  نحوه رمزگشایی و خواندن پیام
 متن فارسی دریافتی را در فایل persian_text.txt کپی و ذخیره کنید.
 برنامه را اجرا کنید:
 ```bash
 python secret_message.py
 ``` 
 رمز عبوری که فرستنده به شما گفته را وارد کنید

  گزینه 2 (Decrypt) را انتخاب کنید.

  پیام اصلی در فایل english_text.txt ظاهر خواهد شد.

###  نکات امنیتی

امنیت دو مرحله‌ای: حتی اگر کسی فایل کلید شما را بدزدد، بدون داشتن رمز عبور نمی‌تواند پیام را بخواند.
